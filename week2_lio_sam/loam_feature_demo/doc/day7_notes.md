# Day7 LOAM特征提取与点到面ICP配准
## 一、核心知识点
### 1. LOAM 激光里程计核心思想
LOAM（Lidar Odometry and Mapping）将激光SLAM前端分为特征提取与位姿求解两步：
- 特征提取：从原始点云中提取**平面点**与**角点（边缘点）**两类高区分度特征
- 位姿求解：平面点使用点到面ICP残差，角点使用点到线ICP残差，联合优化求解位姿
优势：降低点云数据量，提升配准速度与精度，是工业级激光SLAM的主流前端方案。

### 2. 特征提取原理
基于邻域粗糙度判断特征类型：
- 平面点：邻域内点分布平整，拟合平面误差小，粗糙度低于阈值
- 角点：邻域内点离散度高，位于棱边/墙角处，粗糙度高于阈值
工程实现通常基于扫描线的邻域点计算曲率，本实验简化为KD树最近邻标准差计算。

### 3. 点到面ICP原理
- 点到点ICP：最小化源点与目标对应点的欧氏距离，易受点云分布影响，收敛慢
- 点到面ICP：最小化源点到目标平面的垂直距离，利用平面法向约束，收敛更快、精度更高
残差公式：$r = \mathbf{n}^T \cdot (R\mathbf{p} + \mathbf{t} - \mathbf{q})$
其中$\mathbf{n}$为目标平面法向量，$\mathbf{p}$为源点，$\mathbf{q}$为目标平面上的对应点。

### 4. 优化实现
使用Ceres自动微分求解非线性最小二乘问题，优化变量为6维位姿：
- 3维旋转向量（罗德里格斯公式转换为旋转矩阵）
- 3维平移向量
求解器采用DENSE_QR，适合小规模优化问题。

## 二、实验内容
1. 生成结构化模拟点云（地面+两面墙），模拟室内墙角场景
2. 实现基于邻域粗糙度的LOAM风格特征提取，输出平面点与角点两类特征
3. 基于Ceres实现点到面ICP配准，对比估计位姿与真值
4. launch一键启动特征提取演示链路
5. RViz可视化特征提取效果

## 三、实验结果
- 点到面ICP在理想平面点云下2轮迭代即可高精度收敛，残差降至10⁻¹⁷量级
- 估计位姿与真值误差在双精度计算极限内，验证了点到面ICP的高精度特性
