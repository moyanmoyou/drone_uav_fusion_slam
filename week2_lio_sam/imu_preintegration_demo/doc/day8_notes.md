# Day8 IMU预积分理论与基础实现
## 一、核心知识点
### 1. IMU测量模型
IMU包含加速度计与陀螺仪两类传感器，测量值均包含真实值、零偏与高斯噪声：
- 加速度计：测量机体坐标系下的**比力**（包含重力加速度与运动加速度）
  $$\tilde{a}_b = R_{wb}^T (a_w + g_w) + b_a + n_a$$
- 陀螺仪：测量机体坐标系下的角速度
  $$\tilde{\omega}_b = \omega_b + b_g + n_g$$
> 关键注意：加速度计无法区分重力与运动加速度，积分时必须扣除重力项才能得到真实运动加速度。

### 2. IMU预积分核心思想
传统IMU积分依赖初始位姿，位姿优化时每次更新初值都需要重新积分所有IMU数据，计算效率低。
预积分直接计算两帧时刻之间的**相对运动增量**（相对位置、相对速度、相对姿态），与初始位姿无关，位姿优化时无需重复积分，是LIO紧耦合后端优化的核心基础组件。

### 3. 常用数值积分方案
- 欧拉积分：仅使用当前帧测量值计算增量，计算简单但精度低
- 中值积分：使用相邻两帧测量值的平均值计算增量，精度更高，是工业界主流方案
- 龙格-库塔积分：多阶迭代，精度最高但计算复杂度高，多用于高精度惯导场景

## 二、实验内容
1. 实现IMU数据模拟器，生成带零偏与高斯噪声的100Hz IMU测量数据
2. 基于中值积分实现简化版IMU预积分，支持相对位置、速度、姿态增量计算
3. 扣除重力项，还原真实运动加速度
4. 每秒输出一次预积分结果，与匀加速运动真值对比验证精度

## 三、实验结果
- 预积分位置误差1毫米级，速度误差0.001m/s级，误差完全来自高斯噪声
- 重力扣除生效，Z轴无累积漂移
- 验证了中值积分方案的精度与稳定性

## 四、后续进阶方向
当前为入门简化版，完整工业级预积分还需补充：
1. 协方差传递：计算预积分增量的不确定度
2. 零偏建模：将零偏作为优化变量，支持零偏更新后的预积分修正
3. 连续时间积分：适配非等间隔IMU数据
